## 压缩文件

### zip

压缩文件

```bash
zip -r <name>.zip /<path>
```

解压文件

```bash
unzip <name>.zip
```

### tar

压缩文件

```bash
tar -zcvf <name>.tar.gz /<path>
```

解压文件

```bash
tar -zxvf <name>.tar.gz
```
