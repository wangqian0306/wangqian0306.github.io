Windows 打开 WSL2

简介

WSL 是 windows推出的可让开发人员不需要安装虚拟机或者设置双系统启动就可以原生支持运行 GNU/Linux 的系统环境，简称 WSL 子系统。

开启 wsl

  • 打开控制面板(大图标查看方式)
  • 打开程序与功能选项
  • 在左侧菜单栏选择 ‘启用或关闭 Windows 功能’
    • 选中 ‘虚拟机平台’
    • 选中 ‘适用于 Linux 的 Windows 子系统’
    • 点击确定即可

注：之后会进行安装，按照提示进行重启即可。
