Windows 平台的注意事项

  • Windows Server 和 Windows 专业版系统上可以使用 Windows 自己的 Kerberos (AD)
  • Windows 平台的 Kerberos 与 MIT Kerberos (Linux) 并不是完全兼容的，如果有在 Windows 平台上开发应用程序的需求请选用 Windows AD。
  • MIT Kerberos 提供了 Windows 版的客户端，支持浏览器调取 Kerberos 认证。
